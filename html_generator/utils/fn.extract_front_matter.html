pub fn extract_front_matter(content: &str) -> Result<String>
Expand description

Extracts front matter from Markdown content.

This function removes the front matter (if present) from the given content and returns the rest of the content. If no front matter is present, it returns the original content.

The front matter should be in the following format:

---
key1: value1
key2: value2
---
